# Mobile Source

This folder contains the Flutter app source code.

## Current State

The source tree is transitional:

- `main.dart` still contains the default scaffold app
- `app/`, `data/`, and `services/` show intended structure
- not every folder is fully wired into a working user flow yet

## Subfolders

- `app/`: app-level shell and routing placeholders
- `data/`: local, remote, and repository layers
- `services/`: operational helpers and experiments
